tree-sitter-zig
================

Zig grammar for [tree-sitter].

### Development

Install the dependencies:

    npm install

Run the tests:

    npm run test

Run the build and tests in watch mode:

    npm run test:watch

### Git Submodules

* [zig-spec] - provides Zig file samples used to test this parser.
